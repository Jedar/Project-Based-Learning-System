# Angular-cli常用命令  

## 列表  

|命令|描述|
|---|---|
|ng new [project]|创建新项目|
|ng g module [module name]|创建新模块|
|ng g component --module=[xxx.module.ts] [component name]|创建组件，需要指定组件所属的模块,注意需要进入到指定模块的目录下|
