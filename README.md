# Project-based Learning

## 任务描述 
PBL系统结合项目是学习的有关研究，将项目与课程相结合，学生通过在线课程学习，自由
组队，并从感兴趣的项目出发，而不只是单调的听课，来激发学习兴趣，并锻炼动手能力、团队协作能
力。本项目的开发目标是结合PBL思想开发一个类似超星的学习平台。

## 项目技术栈
|模块|技术|
|---|---|
|前端|Angular|
|后端框架|Spring boot|
|持久化|Mybatis|
|数据库|Mysql|
